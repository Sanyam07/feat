FEW-two

Few-two is a feature engineering wrapper that wraps around the Shogun C++ machine learning toolkit and interfaces with scikit-learn. Its purpose is to learn new representations of raw data to improve classifier and regressor performance. The underlying methods are based on Pareto optimization and evolutionary computation to search the space of possible transformations.

FEW-two is a completely different code base from Few. The main differences are:

  • Each individual in FEW-two is its own ML model + data representation, instead of one piece of an overall model
  • FEW-two is pure c++
  • FEw-two uses the Shogun C++ ML toolbox instead of Scikit-learn

Install

Few-two depends on the Eigen matrix library for C++ as well as the Shogun ML library.

Ubuntu

It's easy to install the Eigen deb package:

1 sudo apt-get install libeigen3-dev

And the shogun package:

1 sudo add-apt-repository ppa:shogun-toolbox/stable
2 sudo apt-get update
3 sudo apt-get install libshogun18

Few-two uses cmake to build. So clone the repo, create a build directory, go into it and use cmake to generate the make file. Then you can build by calling make.

1 git clone https://github.com/lacava/fewtwo
2 cd fewtwo
3 mkdir build
4 cd build
5 cmake ..
6 make
